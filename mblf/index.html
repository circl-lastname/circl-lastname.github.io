MBLF and Brainfuck stuff

In order to simplify Brainfuck development, I made a language called MBLF. Right now it's sadly on hiatus.

If you would like to learn how to code with it, visit the wiki. Please don't use the master branch, rather download it using the instrctions provided.

Here is a list of things I would like to some day improve about it:

  • Fix #include. Currently, referencing a variable in an if statement that's defined outside of the current file will cause an error.
  • Generate more efficient Brainfuck code, when adding or subtracting.
  • Stop converting between variable names and addresses in the code so much.
  • Rewrite in anything but JavaScript

If you have any problems with the language feel free to contact me.
