6502 stuff

The 6502 is a nice little chip. I forked an online 6502 simulator, and I enjoyed messing around with it.

I use vasm as my assembler, and a node script I made to convert the binary data to dcb statements for the online simulator.

Writing a basic command line interface and a simple paint app using the built-in screen has introduced me to low-level assembly coding, and ultimately helped me understand C and lower-level programming in general.
